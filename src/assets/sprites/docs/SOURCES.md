- Source page: https://opengameart.org/content/pixel-art-character
- Author: acasas
- License shown on source page: CC0
- Included project file: `src/assets/sprites/agents/office_human_front_walk.png`
- Note: this is a processed strip cropped from `Boy  sheet wlaking and Idle.png`, using the front-facing walk/idle frames only.
