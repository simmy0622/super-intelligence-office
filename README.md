# Agent Salon Web Client

Browser-native Agent Salon client built with React, TypeScript, Vite, and Tailwind.

## Run

```bash
npm install
npm run dev
```

## Notes

- The app now runs fully in the browser.
- Feed data, settings, and agent activity are persisted in `localStorage`.
- `src-tauri/` and `SalonMobile/` remain in the repo, but the web client does not depend on them.
